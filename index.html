Selected visualizations

Fig. 01

Elephant movement routes

GPS tracker data from a population of elephants, processed and mapped in Python with the Folium package. Each line is one individual's recorded track; where many tracks overlap, the brighter corridors emerge — showing the routes and core areas the herd relies on most.

Geoscripting · Python, Folium
Fig. 02

NDVI of Harris County, Houston

Normalized Difference Vegetation Index calculated from open-source satellite imagery and processed in ArcGIS. Green represents high NDVI and red represents low NDVI, highlighting where urban green space is present — and where it is lacking — across Harris County.

Geo-Information Science in Context · ArcGIS
Fig. 03

Pipeline route suitability, Maas-Waal

A multi-criteria suitability analysis built in ArcGIS ModelBuilder to find the best route for a wastewater pipeline connecting industrial wastewater locations to a refinement basin in the Maas-Waal area. The upper map shows the suitability surface and resulting optimal path; the lower map places that path in its real-world context alongside the source locations and the basin.

Geo-Information Tools · ArcGIS ModelBuilder
Fig. 04

Land cover classification, Oosterbeek

A K-nearest neighbours classifier trained in Python on Sentinel-2 imagery to map land cover around Oosterbeek — distinguishing built-up areas, forest, herbaceous vegetation, and water. The result (right) is shown next to the S2-GLC reference product, used as ground truth to assess the classifier's accuracy along the urban–rural gradient.

Machine Learning · Python, scikit-learn
